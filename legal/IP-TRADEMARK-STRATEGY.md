## 3. SOUTH AFRICAN TRADEMARK REGISTRATION PROCESS

### 3.1 Registration Authority

**Companies and Intellectual Property Commission (CIPC)**
- Website: www.cipc.co.za
- Email: trademarks@cipc.co.za
- Physical: DTI Campus, 77 Meintjies Street, Sunnyside, Pretoria

### 3.2 Trademark Classes (Nice Classification)

For SiyaBusa ERP, we need to register in the following classes:

| Class | Description | Applicable To |
|:------|:------------|:--------------|
| **Class 9** | Computer software, downloadable software, mobile apps | SiyaBusa ERP software |
| **Class 35** | Business management software, business consulting | ERP services |
| **Class 42** | Software as a Service (SaaS), cloud computing, IT consulting | Cloud ERP platform |

**Recommendation:** Register in Classes 9, 35, and 42 for comprehensive protection.

### 3.3 Registration Process

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TRADEMARK REGISTRATION TIMELINE                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  STEP 1: Search (1-2 weeks)                                         │
│  └─> Conduct trademark search on CIPC database                      │
│  └─> Check for similar marks in Classes 9, 35, 42                   │
│                                                                      │
│  STEP 2: Application (1 day)                                        │
│  └─> Complete Form TM1                                              │
│  └─> Pay application fee                                            │
│  └─> Submit to CIPC                                                 │
│                                                                      │
│  STEP 3: Examination (3-6 months)                                   │
│  └─> CIPC examines application                                      │
│  └─> Respond to any objections                                      │
│                                                                      │
│  STEP 4: Publication (2 months)                                     │
│  └─> Mark published in Patent Journal                               │
│  └─> 3-month opposition period                                      │
│                                                                      │
│  STEP 5: Registration (1-2 months)                                  │
│  └─> Certificate issued if no opposition                            │
│  └─> Effective from application date                                │
│                                                                      │
│  TOTAL TIMELINE: 9-18 months                                        │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## 5. COPYRIGHT PROTECTION

### 5.1 Automatic Copyright

Under the South African Copyright Act 98 of 1978, copyright is automatic and covers:

| Work | Protection | Duration |
|:-----|:-----------|:---------|
| Software source code | Literary work | 50 years from creation |
| Database schemas | Literary work | 50 years |
| User interface designs | Artistic work | 50 years |
| Documentation | Literary work | 50 years |
| Marketing materials | Literary/Artistic | 50 years |

### 5.2 Copyright Notice

Include on all materials:

```

SiyaBusa®, SARS Sentinel™, and associated logos are trademarks of Masaphokati Technologies (Pty) Ltd.
```
