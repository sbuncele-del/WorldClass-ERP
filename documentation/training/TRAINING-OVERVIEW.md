# SiyaBusa ERP - Training Program Overview

## Comprehensive Learning Path for All Users

## Training Philosophy

At SiyaBusa, we believe effective training is:
- **Role-Based** - Learn what you need for your job
- **Practical** - Focus on real tasks, not theory
- **Progressive** - Build skills step-by-step
- **Supported** - Help available when you need it

## Certification Program

### How It Works

1. **Complete Training** - Finish all required modules
2. **Pass Assessment** - Score 80%+ on online test
3. **Receive Certificate** - Digital certificate issued
4. **Maintain Status** - Annual refresher required

### Certificate Levels

| Level | Requirements | Benefits |
|:------|:-------------|:---------|
| **Bronze** | End User Path | Basic system access |
| **Silver** | Path + 1 Module | Priority support |
| **Gold** | Path + 3 Modules | Advanced features |
| **Platinum** | All Modules + Admin | Full access + consulting |

### Assessment Format

- 30-50 multiple choice questions
- 45 minutes time limit
- Immediate results
- Can retake after 24 hours
- Questions randomized

## Frequently Asked Questions

**Q: How long does full training take?**
A: Basic proficiency: 2-4 hours. Full certification: 8-12 hours depending on role.

**Q: Can I skip modules I already know?**
A: Yes, you can take the assessment directly. Score 80%+ to earn the certificate.

**Q: Is training mandatory?**
A: End User training is required for system access. Additional training is recommended but optional.

**Q: How often do I need to recertify?**
A: Annually, or when major features are released.

**Q: Can I train my team internally?**
A: Yes! Train-the-trainer programs available. Contact us for details.
