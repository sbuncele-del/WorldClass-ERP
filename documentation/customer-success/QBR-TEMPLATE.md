## QBR Overview

### What is a QBR?

A Quarterly Business Review (QBR) is a strategic meeting with customers to:
- Review achievements and value delivered
- Discuss upcoming business priorities
- Align on success metrics and goals
- Identify expansion and optimization opportunities
- Strengthen the business relationship

### QBR Objectives

| Objective | Purpose |
|-----------|---------|
| **Demonstrate Value** | Show ROI and outcomes achieved |
| **Deepen Relationship** | Engage executive sponsors |
| **Identify Risks** | Surface concerns before they escalate |
| **Drive Adoption** | Highlight underutilized features |
| **Expand Partnership** | Introduce relevant new capabilities |
| **Plan Forward** | Align on goals for next quarter |

### Who Should Attend

| SiyaBusa Team | Customer Team |
|---------------|---------------|
| Customer Success Manager | Executive Sponsor |
| Account Executive (optional) | Finance/Operations Lead |
| Product Manager (for major accounts) | IT Lead |
| Support Lead (if issues exist) | Key Users |

## Pre-QBR Preparation

### 4 Weeks Before QBR

- [ ] Schedule meeting with all key stakeholders
- [ ] Send calendar invite with agenda preview
- [ ] Request any specific topics from customer

### 2 Weeks Before QBR

- [ ] Pull usage analytics and metrics
- [ ] Review support ticket history
- [ ] Check health score and trends
- [ ] Identify achievements and success stories
- [ ] Note any underutilized features
- [ ] Prepare expansion recommendations
- [ ] Draft presentation deck

### 1 Week Before QBR

- [ ] Finalize presentation deck
- [ ] Internal review with team
- [ ] Send reminder to customer
- [ ] Confirm attendees

### Day Before QBR

- [ ] Test presentation/screen sharing
- [ ] Review customer's latest news
- [ ] Prepare backup materials
- [ ] Set up recording (if permitted)

## QBR Best Practices

### Do's ✅

1. **Prepare thoroughly** - Know the customer's data inside out
2. **Lead with value** - Start with achievements, not problems
3. **Listen more than talk** - QBR is about the customer
4. **Be specific** - Use actual numbers and examples
5. **Document everything** - Capture notes and commitments
6. **Follow up promptly** - Send summary within 24 hours
7. **Make it actionable** - End with clear next steps

### Don'ts ❌

1. **Don't wing it** - Unprepared QBRs damage credibility
2. **Don't hide problems** - Address issues proactively
3. **Don't oversell** - QBR is about value, not sales pitch
4. **Don't skip metrics** - Data makes the value tangible
5. **Don't monopolize** - Let customer speak 40%+ of time
6. **Don't forget follow-up** - Action items without follow-up are worthless
