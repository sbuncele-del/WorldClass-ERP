## 8. Change Management

### 8.1 Change Request Process

Changes to this SOW must follow this process:

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  Change         │────▶│  Impact         │────▶│  Approval       │
│  Request        │     │  Assessment     │     │  (Both Parties) │
└─────────────────┘     └─────────────────┘     └─────────────────┘
         │                      │                       │
         ▼                      ▼                       ▼
    Submit CR Form         1-3 Business Days      Written Sign-off
                           Scope, Cost, Time      Required
```

### 8.2 Change Request Form

Changes will be documented using a Change Request Form including:

- Change description
- Business justification
- Scope impact
- Cost impact
- Schedule impact
- Approval signatures

### 8.3 Change Approval Authority

| Change Impact | Approval Required |
|---------------|-------------------|
| < R10,000 / < 1 week | Project Leads (both parties) |
| R10,000 - R50,000 / 1-4 weeks | Executive Sponsors |
| > R50,000 / > 4 weeks | Contract Amendment |
