## 5. Data Security

### 5.1 Security Measures

We implement appropriate technical and organizational measures to protect personal information:

**Technical Measures:**
| Measure | Description |
|---------|-------------|
| **Encryption at Rest** | AES-256 encryption for stored data |
| **Encryption in Transit** | TLS 1.3 for all data transfers |
| **Access Controls** | Role-based access, multi-factor authentication |
| **Network Security** | Firewalls, intrusion detection, DDoS protection |
| **Backup & Recovery** | Daily encrypted backups, disaster recovery |
| **Monitoring** | 24/7 security monitoring, logging |

**Organizational Measures:**
| Measure | Description |
|---------|-------------|
| **Access Management** | Least privilege principle, access reviews |
| **Employee Training** | Security awareness, data protection training |
| **Incident Response** | Documented procedures, breach notification |
| **Vendor Management** | Security assessments, contractual controls |
| **Audits** | Regular security assessments |

### 5.2 Multi-Tenant Security

SiyaBusa ERP uses a multi-tenant architecture with:
- Logical data separation between tenants
- Tenant-specific encryption keys
- Access controls preventing cross-tenant access
- Isolated tenant environments

### 5.3 Incident Response

In the event of a security incident:
1. We will contain and investigate the incident
2. We will notify affected parties as required by POPIA
3. We will report to the Information Regulator within 72 hours if required
4. We will take steps to prevent future incidents

## 7. Your Rights Under POPIA

### 7.1 Your Rights

As a data subject under POPIA, you have the following rights:

| Right | Description |
|-------|-------------|
| **Access** | Request a copy of your personal information |
| **Correction** | Request correction of inaccurate or incomplete information |
| **Deletion** | Request deletion of your personal information |
| **Objection** | Object to processing based on legitimate interests |
| **Restriction** | Request restricted processing in certain circumstances |
| **Portability** | Receive your data in a structured, machine-readable format |
| **Withdraw Consent** | Withdraw consent where processing is based on consent |
| **Lodge Complaint** | File a complaint with the Information Regulator |

### 7.2 How to Exercise Your Rights

To exercise your rights:

1. **Email:** privacy@siyabusa.com
2. **Form:** Complete our Data Subject Request Form at www.siyabusa.com/privacy/request
3. **Mail:** Information Officer, Masaphokati Technologies, [Address]

We will:
- Respond within 30 days
- Verify your identity before processing requests
- Provide information free of charge (reasonable fees may apply for excessive requests)

### 7.3 Identity Verification

To protect your privacy, we verify identity before processing requests through:
- Email verification
- Account authentication
- Government ID (for sensitive requests)

### 7.4 Limitations

We may decline requests where:
- We cannot verify your identity
- The request is excessive or manifestly unfounded
- Legal obligations require retention
- Data is necessary for legal claims
