## 🎯 **Video Production Notes**

### **Technical Specifications:**
- **Resolution:** 1920×1080 (Full HD)
- **Frame Rate:** 30 fps
- **Format:** MP4 (H.264)
- **Audio:** AAC 192kbps stereo
- **Subtitles:** SRT file included for accessibility

### **Recording Setup:**
1. **Screen Recording:** Use OBS Studio or ScreenFlow
2. **Browser:** Chrome in Incognito mode (clean, no extensions)
3. **Cursor:** Medium size, highlight clicks
4. **Zoom:** Use for emphasis on key features
5. **Transitions:** Smooth fade/slide (0.5 sec)

### **Voice Recording:**
- **Microphone:** Studio quality (Blue Yeti or similar)
- **Tone:** Professional but enthusiastic
- **Pace:** 140-160 words per minute
- **Energy:** High at start, steady throughout, crescendo at CTA

### **Background Music:**
- **Style:** Corporate, uplifting, modern
- **Volume:** -20dB (subtle background)
- **Tracks:** Royalty-free from Epidemic Sound or Artlist
- **Suggested:** "Inspiring Corporate" or "Technology Innovation"

### **Text Animations:**
- **Font:** SF Pro Display / Segoe UI (consistent with app)
- **Size:** Large enough for mobile viewing
- **Duration:** 2-3 seconds per text block
- **Effect:** Fade in from bottom

### **Color Grading:**
- Slightly increase saturation (+10%)
- Boost contrast for screen clarity
- Warm white balance (5800K)

### **Export Settings:**
- **YouTube:** 1920×1080, 12 Mbps
- **LinkedIn:** 1920×1080, 10 Mbps  
- **Email/Embed:** 1280×720, 5 Mbps (smaller file)
